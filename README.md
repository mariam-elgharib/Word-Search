# Word-Search
A fun and interactive word search puzzle game built with Python and Tkinter. Players can choose from multiple categories and difficulty levels, search for hidden words in a grid of letters, and track their score. The game includes save/load functionality so that you can resume your progress anytime.

# Installation

# How to Play
1. Enter your name when prompted.
2. If a saved game exists for that name, you'll be asked whether to resume it.
3. Choose a category (f for fruits & vegetables, a for animals, p for programming).
4. Choose a level (1, 2, or 3).
5. The grid will appear with letters. The words to find are listed on the right.
6. Click letters to select them in order (they will highlight yellow). Click again to deselect.
7. After selecting a word, click the Check Word button.
- If correct, the word's letters turn green, are disabled, and the word in the list turns grey. Your score increases.
- If incorrect, the selection clears and you can try again.
8. Use the Save Game button to save your progress at any time.
9. Press Escape to exit. You'll be asked whether to save your score.
